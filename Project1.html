Overview

This portfolio website serves as both a personal project and a professional showcase. It highlights my technical skills in **HTML, website deployment, and security**.

Key Features

  • HTML & CSS Foundation: Built from scratch using clean, semantic code.
  • GitHub Pages Deployment: Hosted using GitHub Pages with version control.
  • Security Measures: Implements **Content Security Policy (CSP)** to prevent attacks.
  • Scalability: Plans to transition hosting, improve security, and add more features.

Future Enhancements

Planned improvements include:

  • Enhancing **security** with HTTPS enforcement and better authentication.
  • Adding **dynamic content** using JavaScript or a backend framework.
