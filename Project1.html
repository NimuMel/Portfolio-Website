Project Overview

This portfolio website was created to showcase my knowledge of HTML, CSS, and web hosting.

Goals of this project:

  • Demonstrate proficiency in HTML and CSS.
  • Improve security and reliability through hosting upgrades.
  • Implement security measures such as Content Security Policy (CSP).
  • Use the site to display ongoing and completed projects.

Project Evolution

This website initially started as a simple project hosted on GitHub Pages, providing an easy and free way to showcase my work.

As I needed more flexibility and control, I transitioned the site to an AWS EC2 instance. This allowed for improved security, better customization, and the ability to configure a firewall to protect the machine.

To enhance performance and security further, I integrated Cloudflare for CDN and DDoS protection. Finally, I set up Apache to manage hosting efficiently, allowing me to serve the website on my custom domain, nimuemeller.dev.
