Project Overview

This project focuses on developing a secure API to facilitate communication between IoT devices while ensuring data integrity and protection against cyber threats.

Goals of this project:

  • Design a RESTful API with authentication and encryption.
  • Ensure secure device-to-cloud communication.
  • Implement role-based access control (RBAC) for users and devices.
  • Test the API against common security vulnerabilities.
  • Deploy using a cloud-based hosting solution.
